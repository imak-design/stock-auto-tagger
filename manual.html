1. システム概要

ストック素材（画像・動画・ベクター EPS）を 3 つのストックフォトサイトに一括登録するための自動化ツールです。

  • Gemini AI で画像・動画を解析し、英語/日本語のタイトル・キーワード・カテゴリを自動生成
  • Playwright でブラウザを自動操作し、ファイルアップロード → メタデータ適用 → 審査提出まで全自動
  • GUI（Tkinter）でボタンを押すだけで操作可能

対応サイトとファイル形式

サイト画像動画ベクター
Adobe StockJPG / PNGMP4 / MOVEPS
ShutterstockJPGMP4 / MOVEPS
PixtaJPG / PNGMP4 / MOVZIP (EPS+PNG)

※ Shutterstock は PNG 非対応のため、JPG と動画のみアップロードされます。

4. セットアップ

Step 1: Python のインストール

Mac

# Homebrew をインストール（まだの場合）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Python をインストール
brew install python

Windows

python.org からインストーラーをダウンロード。
インストール時に 「Add Python to PATH」にチェック を入れてください。

Step 2: Git のインストール（Windows のみ）

git-scm.com からインストーラーをダウンロードし、デフォルト設定のままインストール。

Git を入れたくない場合は、GitHub ページの緑色の「Code」ボタン → 「Download ZIP」からダウンロードしてください。

Step 3: リポジトリのダウンロード

git clone https://github.com/imak-design/stock-auto-tagger.git
cd stock-auto-tagger

Step 4: 依存パッケージのインストール

Mac

python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
brew install python-tk@3.14
playwright install chromium

Windows

pip install -r requirements.txt
playwright install chromium

Step 5: 設定ファイルを作成

cp config.example.json config.json

config.json を開いて設定を入力：

{
  "input_folder": "/Users/あなた/Desktop/stock_input",
  "variation_folder": "",
  "backup_folder": "",
  "enabled_sites": ["adobe", "shutterstock", "pixta"]
}
項目説明
input_folder素材を置くフォルダ（JPG/PNG/MP4 を入れる）
variation_folderバリエーション素材フォルダ（工程0用、不要なら空欄）
backup_folderバックアップ先（不要なら空欄）
enabled_sitesアップロードするサイト（不要なサイトを配列から削除）
test_modetrue で審査申請をスキップ（GUI からも切替可）

5. API キーの設定

API キーはプロジェクトフォルダの外に保存します。フォルダを共有しても API キーが漏れる心配がありません。
OS保存先パス
Mac~/.stock-auto-tagger/.env
WindowsC:\Users\ユーザー名\.stock-auto-tagger\.env

.env ファイルの中身：

GEMINI_API_KEY=あなたのAPIキー

作成コマンド

Mac

mkdir -p ~/.stock-auto-tagger
echo "GEMINI_API_KEY=ここにキーを貼付" > ~/.stock-auto-tagger/.env

編集: open -e ~/.stock-auto-tagger/.env

Windows

mkdir %USERPROFILE%\.stock-auto-tagger
echo GEMINI_API_KEY=ここにキーを貼付 > %USERPROFILE%\.stock-auto-tagger\.env

編集: notepad %USERPROFILE%\.stock-auto-tagger\.env

API キーの取得

Google AI Studio で無料取得できます。

無料枠の目安（Gemini 2.5 Flash）

無料 API キーは 1日25リクエスト（1分5リクエスト）の上限があります。画像1枚 = 1リクエスト。

モード1日あたり
工程1のみ（タグ生成）約 20 枚
工程0（リネーム + タグ生成）約 10 枚

上限は太平洋時間の深夜0時（日本時間 17:00）にリセットされます。

API キーの取り扱い注意
  • API キーは必ず上記の .env ファイルに書いてください
  • config.json など他のファイルには書かないでください
  • AI チャットに API キーを貼り付けると会話ログに残る可能性があります

6. サイトログイン

使いたいサイトだけ実行してください。ブラウザが開くので手動でログインすると、セッションが自動保存されます。

python adobe_login.py        # Adobe Stock
python shutterstock_login.py  # Shutterstock
python pixta_login.py         # Pixta

セッションが切れた場合は再度実行してください。

8. フォルダ構成と振り分け

input_folder/
├── image1.jpg            ← 通常素材
├── image2.png
├── video1.mp4
├── AI/                   ← AI 生成素材
│   └── ai_image.png
├── Photo/                ← 写真素材
│   └── DSC01234.jpg
├── Vector/               ← ベクター素材
│   └── 260402_icon/
│       ├── 260402_icon.eps
│       └── 260402_icon.png
└── csv_output/           ← CSV 自動出力（自動作成）

AI / Photo フォルダによる自動振り分け

フォルダAdobe StockShutterstockPixta
通常（直下）イラスト通常イラストページ
Photo/写真カテゴリ通常写真ページ
AI/イラスト + AI チェック対象外イラスト + AI チェック

フォルダがなければ全ファイルが通常素材として処理されます。Shutterstock は AI 作品登録不可のため AI フォルダは除外されます。

9. テストモード

GUI の「テストモード」チェックボックスを ON にすると、各サイトで 審査申請の手前で処理が停止 します。

サイト停止位置
Adobe StockCSV 適用 → コンテンツタイプ設定 → 全選択完了 → 「審査に登録」ボタン手前
ShutterstockCSV 適用完了 → 提出ボタン手前
PixtaAI チェック → 全選択完了 → 「選択した作品を登録」ボタン手前
処理完了 → ブラウザが開いたまま停止
↓
タイトル・タグを手動で確認・修正
↓
ブラウザを閉じる → 自動的に次のサイトへ
  • テストモード時はファイル移動もスキップ → 同じ素材で繰り返しテスト可能
  • 工程0（一気通貫）ではブラウザを閉じると次のサイトの処理に自動的に進みます

10. カスタマイズ

プロンプトの編集

prompts/ フォルダ内のテキストファイルを編集するだけで、AI のタグ生成ルールを変更できます。Python コードを触る必要はありません。

ファイル用途
prompts/image_prompt.txt画像用プロンプト
prompts/video_prompt.txt動画用プロンプト

Claude でカスタマイズ

Claude（AI アシスタント）にコードを見せて、やりたいことを伝えるだけでカスタマイズできます。

  • 「Adobe Stock だけ使いたいので、他のサイトの処理を削除して」
  • 「動画は扱わないので、画像だけのシンプル版にして」
  • 「タグの生成ルールを変えたい」

11. トラブルシューティング

症状原因対処
Session expired サイトのセッション切れ 該当の *_login.py を再実行
Gemini 429 エラー API レート制限 自動リトライで回復待ち（無料枠: 1分5リクエスト）
Pixta タグ 0 個 Gemini レスポンス異常 ログを確認し、プロンプトを調整
アップロードが途中で停止 セッション期限切れ 該当サイトに再ログイン
API キーが見つからない .env ファイルが未作成 API キーの設定 を参照
ブラウザが自動で閉じない
（テストモード）		 正常動作 確認後、手動でブラウザを閉じてください
CSV 適用後にタグが空 ポータル反映遅延 自動リトライで待機。解消しない場合は手動でリロード
