# 旧バージョンをダウンロードした方へ

2026年4月15日の更新で、機密ファイルの保存場所を**プロジェクトフォルダ外**に変更しました。

## 旧版をお使いの方へ（重要）

それ以前にクローン・ダウンロードした方は、プロジェクトフォルダ内に以下の機密情報が保存されています：

- `config.json`
- セッションファイル（Adobe/Shutterstock/Pixtaのログイン情報）
- ブラウザプロファイル

**このプロジェクトフォルダをZIP化・共有・公開リポジトリへpushしないでください。** アカウントを乗っ取られる恐れがあります。

（`.env`は旧版からホームディレクトリ保存なので対象外です）

## 最新版に更新したい方へ

ファイル配置が変わっているため、単純に `git pull` すると設定・セッションが読み込まれなくなります。Claude Codeに以下のように頼んでください：

> stock-auto-taggerを最新版に更新して。旧版の `config.json` / セッションファイル / ブラウザプロファイルを `~/.stock-auto-tagger/`（Windowsは `C:\Users\ユーザー名\.stock-auto-tagger\`）に移動してほしい。自分でカスタマイズしたコードがあれば残して。
